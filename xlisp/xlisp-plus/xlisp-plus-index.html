Xlisp Plus Functions

  • debug, nodebug - turn the break loop on and off for errors
  • defconstant - define a pseudo 'constant' variable
  • defparameter - define a pseudo 'parameter' variable
  • defvar - define a variable without overwriting an existing one
  • get-macro-character - return the code that will be executed when a specified character is read
  • set-macro-character - install the code that will be executed when a specified character is read
  • makunbound - make a symbol's value unbound
  • fmakunbound - make a symbol's function definition unbound
  • mapcan - destructive version of mapcar
  • mapcon - destructive version of maplist
  • savefun - save a function or macro to a file
  • pp - advanced version of pprint

Note: The 'XLISP Plus functions' listed on this page are not a regular part of Nyquist. Most of them try to make XLISP more compatible with Common Lisp. I have copied the XLISP code to the related reference pages so you can decide yourself wether you want to use it or not. To use it just simply copy the code from the screen to your 'init.lsp' file.
