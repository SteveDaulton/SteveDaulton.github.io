+  (variable)

Type:   -   variable
Source:   -   xlinit.c, xlisp.c

Syntax

 +

Description

The '+' variable is set to the most recent input expression.

Examples

(setq hi 'there)  ;returns THERE
+                 ;returns (SETQ HI (QUOTE THERE))
+                 ;returns +

Note: It is best not to use this variable in a program.

See the  +  variable in the XLISP 2.0 manual.
