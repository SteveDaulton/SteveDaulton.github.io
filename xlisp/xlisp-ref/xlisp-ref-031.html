asin

Type:   -   function (subr)
Source:   -   xlmath.c

Syntax

(asin expr)
expr - floating point number/expression
returns - the arcsine of the number

Description

The 'asin' function returns the arc sine of the 'expr'. The result is in radians.

Examples

(asin  0.0)   ; returns  0.0
(asin  1.0)   ; returns  1.5708
(asin -1.0)   ; returns -1.5708
(asin    0)   ; error: bad integer operation

Note: this function is not implemented in Nyquist.

See the asin function in the XLISP 2.0 manual.
